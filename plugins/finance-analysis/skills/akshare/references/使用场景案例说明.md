## 场景 1：股票投资价值分析

**需求**：分析某只股票的投资价值

**接口组合**：
1. `stock_profile_cninfo` - 公司概况
2. `stock_financial_benefit_new_ths` - 利润表
3. `stock_financial_debt_new_ths` - 资产负债表
4. `stock_zh_a_hist` - 历史行情
5. `stock_circulate_stock_holder` - 十大股东
6. `stock_fhps_detail_em` - 分红记录

**代码示例**：

```python
import sys
sys.path.append("scripts")
from robust_fetcher import AKShareFetcher

symbol = "600519"  # 贵州茅台
fetcher = AKShareFetcher(verbose=True)

# 1. 公司概况
info = fetcher.fetch("stock_profile_cninfo", symbol=symbol)
if info is not None:
    print(f"公司名称: {info.iloc[0]['公司名称']}")
    print(f"所属行业: {info.iloc[0]['所属行业']}")

# 2. 利润表（营收增长）
profit = fetcher.fetch("stock_financial_benefit_new_ths", symbol=symbol, indicator="按报告期")
if profit is not None:
    print(f"最新营收: {profit.iloc[0]['营业总收入']}")

# 3. 资产负债表（财务健康）
balance = fetcher.fetch("stock_financial_debt_new_ths", symbol=symbol, indicator="按报告期")
if balance is not None:
    print(f"资产负债率: {balance.iloc[0]['资产负债率']}")

# 4. 历史行情（价格走势）
hist = fetcher.fetch(
    "stock_zh_a_hist",
    symbol=symbol,
    period="daily",
    start_date="20240301",
    end_date="20250301"
)

# 5. 十大流通股东（机构持仓）
top10 = fetcher.fetch("stock_circulate_stock_holder", symbol=symbol)

# 6. 分红记录（股息率）
dividend = fetcher.fetch("stock_fhps_detail_em", symbol=symbol)
```

## 场景 2：两只股票对比分析

**需求**：比较两只股票的优劣

**接口组合**：
- `stock_profile_cninfo` - 公司概况
- `stock_financial_abstract` - 财务指标
- `stock_zh_a_hist` - 历史行情

**代码示例**：

```python
import sys
sys.path.append("scripts")
from robust_fetcher import AKShareFetcher

symbols = ["300750", "002594"]  # 宁德时代 vs 比亚迪
fetcher = AKShareFetcher(verbose=True)

results = {}
for symbol in symbols:
    # 基本信息和公司概况
    info = fetcher.fetch("stock_profile_cninfo", symbol=symbol)

    # 财务指标
    financial = fetcher.fetch("stock_financial_abstract", symbol=symbol)

    # 历史行情
    hist = fetcher.fetch("stock_zh_a_hist", symbol=symbol, period="daily")

    results[symbol] = {
        'info': info,
        'financial': financial,
        'hist': hist
    }
```

## 场景 3：监控股东变化

**需求**：查看股东变动情况

**接口组合**：
- `stock_circulate_stock_holder` - 十大流通股东
- `stock_zh_a_gdhs_detail_em` - 股东户数变化
- `stock_shareholder_change_ths` - 持股变动详情

**代码示例**：

```python
import sys
sys.path.append("scripts")
from robust_fetcher import AKShareFetcher

symbol = "000001"
fetcher = AKShareFetcher(verbose=True)

# 十大流通股东
top10 = fetcher.fetch("stock_circulate_stock_holder", symbol=symbol)

# 股东户数变化
holders = fetcher.fetch("stock_zh_a_gdhs_detail_em", symbol=symbol)

# 持股变动详情
changes = fetcher.fetch("stock_shareholder_change_ths", symbol=symbol)
```

## 场景 4：财务数据分析

**需求**：查看公司财务报表

**接口组合**：
- `stock_financial_debt_new_ths` - 资产负债表
- `stock_financial_benefit_new_ths` - 利润表
- `stock_financial_cash_new_ths` - 现金流量表
- `stock_financial_abstract` - 财务分析指标

**代码示例**：

```python
import sys
sys.path.append("scripts")
from robust_fetcher import AKShareFetcher

symbol = "000001"
fetcher = AKShareFetcher(verbose=True)

# 三大财务报表
balance = fetcher.fetch("stock_financial_debt_new_ths", symbol=symbol, indicator="按报告期")    # 资产负债表
profit = fetcher.fetch("stock_financial_benefit_new_ths", symbol=symbol, indicator="按报告期")      # 利润表
cashflow = fetcher.fetch("stock_financial_cash_new_ths", symbol=symbol, indicator="按报告期") # 现金流量表

# 财务分析指标
indicators = fetcher.fetch("stock_financial_abstract", symbol=symbol)
```

## 通用分析框架

对于任何分析任务，建议按以下框架组织输出：

1. **数据摘要**：核心指标一览表
2. **趋势分析**：时间序列变化
3. **对比分析**：与历史或同行对比
4. **风险评估**：指出潜在风险
5. **结论建议**：给出明确结论

---

## 接口选择原则

1. **优先使用"主要"接口**：从常用接口清单中选择
2. **官方数据优先**：交易所、巨潮资讯等官方来源
3. **多源验证**：关键数据可从多个接口验证
4. **备用方案**：主接口失败时尝试备选接口
