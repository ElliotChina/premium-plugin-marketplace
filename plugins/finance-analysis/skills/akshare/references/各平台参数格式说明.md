## 东方财富（P1 优先级）

**参数命名特征**：
- ✅ `symbol` - 股票代码（6位数字，如 "000001"）
- ✅ `start_date` / `end_date` - 日期范围（格式 "YYYYMMDD"）
- ✅ `period` - 数据周期（"daily"/"weekly"/"monthly"）
- ✅ `adjust` - 复权类型（"" 不复权/"qfq" 前复权/"hfq" 后复权）

**典型接口**：
- `stock_zh_a_hist` - 历史行情
- `stock_individual_info_em` - 个股信息
- `stock_balance_sheet_by_report_em` - 资产负债表

**参数示例**：
```python
# 历史行情
ak.stock_zh_a_hist(
    symbol="000001",      # 股票代码
    period="daily",       # 周期
    start_date="20240101", # 开始日期
    end_date="20241231",   # 结束日期
    adjust=""              # 复权（空字符串=不复权）
)
```

**注意事项**：
- 业绩快报类接口（`stock_yjbb_em`、`stock_lrb_em` 等）**不支持** `symbol` 参数
- 部分财务报表接口可能返回 None，需要错误处理

## 雪球（P3 优先级）

**参数命名特征**：
- ✅ `symbol` - 证券代码（**需带市场前缀**，如 "SH600000"、"SZ000001"）
- ✅ `token` - 认证令牌（通常为 None，可选）
- ✅ `timeout` - 超时参数（通常为 None，可选）

**典型接口**：
- `stock_individual_spot_xq` - 实时行情
- `stock_hot_follow_xq` - 关注排行榜
- `stock_individual_basic_info_xq` - 公司基本信息

**参数示例**：
```python
# 实时行情
ak.stock_individual_spot_xq(
    symbol="SH600000",  # 需带市场前缀（SH/SZ）
    token=None,          # 认证令牌（通常为None）
    timeout=None         # 超时参数
)

# 热度排行（无参数）
ak.stock_hot_follow_xq()
```

**注意事项**：
- ⚠️ 股票代码**必须**带市场前缀：SH（上交所）、SZ（深交所）
- 热度类接口不需要参数，返回多行数据
- 部分接口可能需要翻页，耗时较长（15-20秒）

## 同花顺（P2 优先级）

**参数命名特征**：
- ✅ `symbol` - 股票代码（6位数字）
- ✅ `indicator` - 报告类型（"按报告期"/"按年度"）

**典型接口**：
- `stock_financial_debt_new_ths` - 资产负债表
- `stock_financial_benefit_new_ths` - 利润表
- `stock_financial_cash_new_ths` - 现金流量表
- `stock_financial_abstract_new_ths` - 财务摘要

**参数示例**：
```python
# 财务报表
ak.stock_financial_debt_new_ths(
    symbol="000001",        # 股票代码
    indicator="按报告期"     # 报告类型
)
```

**注意事项**：
- ✅ 同花顺财务接口数据**最全面**，返回多年历史数据
- 参数值支持中文（如 "按报告期"、"按年度"）
- 财务三表接口（`*_new_ths`）已替代旧版本，建议优先使用

## 新浪财经（P2 优先级）

**参数命名特征**：
- ✅ `symbol` - 股票代码（**需带市场前缀**，如 "sh600000"、"sz000001"）
- ✅ `start_date` / `end_date` - 日期范围
- ✅ `adjust` - 复权类型（支持更多选项）

**典型接口**：
- `stock_zh_a_daily` - 历史行情
- `stock_financial_abstract` - 财务指标

**参数示例**：
```python
# 历史行情
ak.stock_zh_a_daily(
    symbol="sh600000",     # 需带市场前缀（小写）
    start_date="20240101",
    end_date="20241231",
    adjust="qfq"            # 前复权
)
```

**注意事项**：
- ⚠️ **市场前缀必须小写**：sh（上交所）、sz（深交所）
- ⚠️ 多次请求**容易封禁 IP**，建议限制请求频率
- ⚠️ 新浪财经接口**建议替换为东方财富**（数据质量更高，访问无限制）

---

## 巨潮资讯（P2 优先级）

**参数命名特征**：
- ✅ `symbol` - 股票代码（6位数字，**不带市场前缀**）
- ✅ 参数最少，通常只需股票代码

**典型接口**：
- `stock_profile_cninfo` - 公司概况
- `stock_dividend_cninfo` - 历史分红
- `stock_hold_num_cninfo` - 股东户数

**参数示例**：
```python
# 公司概况
ak.stock_profile_cninfo(
    symbol="600030"  # 纯数字代码，不带前缀
)
```

**注意事项**：
- ✅ **官方数据源**，公告信息最权威
- 股票代码**纯数字**，不带市场前缀
- 部分接口（如 `stock_hold_num_cninfo`）参数名可能不是 `symbol`

| 平台 | 股票代码格式 | 市场前缀 | 日期参数 | 复权参数 | 特殊参数 |
|------|-------------|----------|----------|----------|----------|
| **东方财富** | 6位数字 | 否 | start_date, end_date | adjust | - |
| **雪球** | 6位数字 | **是（SH/SZ）** | - | - | token, timeout |
| **同花顺** | 6位数字 | 否 | - | - | indicator（中文） |
| **新浪** | 6位数字 | **是（sh/sz小写）** | start_date, end_date | adjust | - |
| **巨潮资讯** | 6位数字 | 否 | - | - | 参数最少 |

## 常见参数格式错误

| 错误 | 原因 | 解决方案 |
|------|------|----------|
| `got an unexpected keyword argument 'symbol'` | 参数名错误 | 查看接口文档确认正确参数名 |
| `'NoneType' object is not subscriptable` | 接口返回 None | 检查网络或尝试备用接口 |
| `invalid literal for int() with base 10: 'i'` | 参数格式错误 | 确认股票代码格式（是否需要前缀） |
| `Connection aborted` | 网络连接中断 | 添加重试机制或切换数据源 |
