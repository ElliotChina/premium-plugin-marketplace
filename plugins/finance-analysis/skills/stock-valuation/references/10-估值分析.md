# 估值分析

估值分析旨在判断公司股票价格的合理区间，通过 PE、PB、PEG、EV/EBITDA 等多种估值方法，结合历史估值分位和同行对比，综合评估股票是高估、低估还是合理估值。估值是投资决策的核心环节，好公司也需要好价格才能成为好投资。

---

## 一、分析目标

1. **计算估值指标**：通过多种估值方法计算公司当前估值水平
2. **历史估值对比**：判断当前估值处于历史什么位置
3. **同行估值对比**：判断公司相对同行业公司的估值水平
4. **综合估值判断**：综合多种方法给出高估/合理/低估的结论

---

## 二、需要采集的数据

### 2.1 基础数据

| 数据项 | 说明 |
|--------|------|
| 当前股价 | 最新收盘价 |
| 总股本 | 公司总股份数量 |
| 总市值 | 股价 × 总股本 |
| 净资产 | 归属于母公司股东权益 |

### 2.2 利润数据

| 数据项 | 说明 |
|--------|------|
| 净利润（TTM） | 最近 12 个月净利润 |
| 净利润（历史） | 过去年度净利润（用于历史 PE 计算） |
| 扣非净利润（TTM） | 最近 12 个月扣非净利润 |

### 2.3 估值历史数据

| 数据项 | 说明 |
|--------|------|
| PE 历史数据 | 过去 5-10 年的 PE 数据 |
| PB 历史数据 | 过去 5-10 年的 PB 数据 |
| PS 历史数据 | 过去 5-10 年的 PS 数据 |

### 2.4 同行数据

| 数据项 | 说明 |
|--------|------|
| 同行公司市值 | 同行业可比公司市值 |
| 同行 PE(TTM) | 同行业可比公司 PE |
| 同行 PB | 同行业可比公司 PB |
| 同行 PS | 同行业可比公司 PS |

### 2.5 数据期间要求

- **历史估值**：至少 5 年数据（用于计算历史分位）
- **同行数据**：最新数据
- **数据优先级**：TTM 数据 > 年报数据

## 三、核心指标与计算方法

### 3.1 PE 估值指标

| 指标 | 计算公式 | 用途 |
|------|---------|------|
| PE(TTM) | 市值 ÷ 最近 12 个月净利润 | 衡量回本年限 |
| PE(静) | 市值 ÷ 上一年度净利润 | 静态市盈率 |
| PE(动) | 市值 ÷ 预测净利润 | 动态市盈率 |
| PE 历史分位 | 当前 PE 在历史中的百分位 | 判断历史位置 |

### 3.2 PB 估值指标

| 指标 | 计算公式 | 用途 |
|------|---------|------|
| PB | 市值 ÷ 净资产 | 衡量市价相对净资产的倍数 |
| PB 历史分位 | 当前 PB 在历史中的百分位 | 判断历史位置 |

### 3.3 PEG 估值指标

| 指标 | 计算公式 | 用途 |
|------|---------|------|
| PEG | PE ÷ 净利润增长率 × 100 | 衡量估值与成长的匹配性 |

### 3.4 PS 估值指标

| 指标 | 计算公式 | 用途 |
|------|---------|------|
| PS(TTM) | 市值 ÷ 最近 12 个月营业收入 | 适合亏损或高增长公司 |
| PS 历史分位 | 当前 PS 在历史中的百分位 | 判断历史位置 |

### 3.5 EV/EBITDA 指标

| 指标 | 计算公式 | 用途 |
|------|---------|------|
| EV（企业价值） | 市值 + 有息负债 - 货币资金 | 衡量收购公司的实际成本 |
| EBITDA | 净利润 + 所得税 + 折旧 + 摊销 | 衡量经营现金流 |
| EV/EBITDA | EV ÷ EBITDA | 适合重资产行业估值 |

### 3.6 其他估值指标

| 指标 | 计算公式 | 用途 |
|------|---------|------|
| PC（市现率） | 市值 ÷ 经营现金流净额 | 衡量估值与现金流的关系 |
| 股息率 | 每股分红 ÷ 股价 × 100% | 衡量分红收益率 |
| 市值/自由现金流 | 市值 ÷ 自由现金流 | 衡量估值与自由现金流的关系 |

## 四、分析逻辑与判断标准

### 4.1 PE 估值分析

**PE 绝对值判断**：

| PE | 评估 | 说明 |
|----|------|------|
| <10 | 极低估值 | 可能被低估，或盈利不可持续 |
| 10-15 | 低估值 | 估值较低 |
| 15-25 | 合理估值 | 估值合理 |
| 25-40 | 高估值 | 估值较高 |
| >40 | 极高估值 | 可能被高估 |

**行业差异参考**：
- 银行：PE 通常 4-8 倍（低增长、高杠杆）
- 白酒：PE 通常 20-40 倍（高毛利、稳定增长）
- 医药：PE 通常 25-50 倍（研发驱动、高增长）
- 科技/半导体：PE 通常 30-60 倍（高增长、高波动）
- 周期股（钢铁、煤炭）：PE 通常 5-15 倍（盈利波动大）
- 公用事业：PE 通常 10-20 倍（稳定、低增长）

**PE 历史分位判断**：

| 历史分位 | 评估 | 说明 |
|---------|------|------|
| <10% | 极低分位 | 估值处于历史底部 |
| 10-30% | 低分位 | 估值处于历史较低位置 |
| 30-70% | 合理区间 | 估值处于历史合理位置 |
| 70-90% | 高分位 | 估值处于历史较高位置 |
| >90% | 极高分位 | 估值处于历史顶部 |

**PE 适用场景**：
- 适用：盈利稳定、非周期性行业
- 不适用：亏损公司、盈利波动大的周期股

### 4.2 PB 估值分析

**PB 绝对值判断**：

| PB | 评估 | 说明 |
|----|------|------|
| <1 | 破净 | 股价低于净资产 |
| 1-2 | 低 PB | 估值较低 |
| 2-4 | 合理 PB | 估值合理 |
| 4-8 | 高 PB | 估值较高 |
| >8 | 极高 PB | 估值很高 |

**行业差异参考**：
- 银行：PB 通常 0.5-1 倍（资产重、ROE 低）
- 钢铁/煤炭：PB 通常 0.8-1.5 倍（周期性强）
- 制造业：PB 通常 2-4 倍
- 消费/医药：PB 通常 4-10 倍（轻资产、ROE 高）
- 科技/互联网：PB 通常 5-15 倍（轻资产、高 ROE）

**PB 历史分位判断**：同 PE 分位判断标准

**PB 适用场景**：
- 适用：重资产行业、周期性行业、银行保险
- 不适用：轻资产行业、科技公司

### 4.3 PEG 估值分析

**PEG 判断**：

| PEG | 评估 | 说明 |
|-----|------|------|
| <0.5 | 极度低估 | 估值远低于成长性 |
| 0.5-0.8 | 低估 | 估值低于成长性 |
| 0.8-1.2 | 合理 | 估值与成长性匹配 |
| 1.2-1.5 | 略高估 | 估值略高于成长性 |
| 1.5-2 | 高估 | 估值高于成长性 |
| >2 | 严重高估 | 估值远高于成长性 |

**注意事项**：
- 增长率应使用预期增长率（未来 3 年）
- 增长率<0 时 PEG 失效
- 增长率>50% 时 PEG 参考意义下降

**PEG 适用场景**：
- 适用：成长股、盈利增长稳定的公司
- 不适用：亏损公司、增长不稳定的公司

### 4.4 PS 估值分析

**PS 绝对值判断**：

| PS | 评估 | 说明 |
|----|------|------|
| <3 | 低 PS | 估值较低 |
| 3-8 | 合理 PS | 估值合理 |
| 8-15 | 高 PS | 估值较高 |
| >15 | 极高 PS | 估值很高 |

**行业差异参考**：
- 零售/贸易：PS 通常 0.5-2 倍（净利率低）
- 制造业：PS 通常 1-4 倍
- 软件/SaaS：PS 通常 5-20 倍（高毛利、高增长）
- 互联网平台：PS 通常 3-10 倍

**PS 适用场景**：
- 适用：亏损但营收增长的公司、早期互联网公司
- 不适用：盈利稳定的成熟公司

### 4.5 EV/EBITDA 分析

**EV/EBITDA 判断**：

| EV/EBITDA | 评估 | 说明 |
|-----------|------|------|
| <8 | 低估值 | 估值较低 |
| 8-12 | 合理估值 | 估值合理 |
| 12-18 | 高估值 | 估值较高 |
| >18 | 极高估值 | 估值很高 |

**行业差异参考**：
- 电信/公用事业：EV/EBITDA 通常 6-10 倍
- 制造业：EV/EBITDA 通常 8-14 倍
- 科技/互联网：EV/EBITDA 通常 12-25 倍

**EV/EBITDA 适用场景**：
- 适用：重资产行业、高负债公司、跨国对比
- 不适用：轻资产行业、金融企业

### 4.6 综合估值分析

**多方法估值综合判断**：

| 估值方法 | 当前值 | 历史分位 | 同行对比 | 结论 |
|---------|--------|---------|---------|------|
| PE | | | | |
| PB | | | | |
| PEG | | - | | |
| PS | | | | |
| EV/EBITDA | | | | |

**综合结论判断**：
- 多数指标低估：综合低估
- 多数指标合理：综合合理
- 多数指标高估：综合高估

### 4.7 估值趋势分析

**估值变化趋势**：
- PE 下降：估值消化或盈利增长快于股价
- PE 上升：估值扩张或盈利增长慢于股价
- 估值扩张 + 盈利增长：戴维斯双击
- 估值收缩 + 盈利下滑：戴维斯双杀

## 七、注意事项

1. **盈利质量**：PE 估值前提盈利真实可靠，需排除非经常损益影响
2. **周期股估值**：周期股 PE 低时可能是盈利高点，不宜简单判断为低估
3. **亏损公司**：亏损公司 PE 失效，应使用 PS 或其他方法
4. **历史分位数**：历史数据应足够长（至少 5 年），否则分位参考意义有限
5. **行业对比**：应选择真正可比的公司，不同细分行业估值差异大
6. **增长率选择**：PEG 应使用预期增长率而非历史增长率
7. **汇率影响**：跨国对比时需考虑汇率因素
8. **特殊因素**：重组、并购等重大事件可能影响估值可比性
