# 公司基本信息分析

公司基本信息分析是股票价值分析的起点，旨在全面了解公司的基础情况、业务构成和经营概况。通过收集和分析公司基本信息，可以建立对公司的初步认知，为后续深入分析奠定基础。

---

## 一、分析目标

1. **了解公司概况**：掌握公司名称、代码、所属行业、上市时间等基础信息
2. **分析业务结构**：了解公司主营业务构成、各业务收入占比和利润贡献
3. **评估主营业务集中度**：判断公司业务是专注还是多元化
4. **识别第二增长曲线**：分析公司是否有新的业务增长点

---

## 二、需要采集的数据

### 2.1 公司基础信息

| 数据项 | 说明 |
|--------|------|
| 公司全称 | 公司法定注册名称 |
| 股票简称 | 证券交易所显示的简称 |
| 股票代码 | 证券交易所分配的代码 |
| 上市交易所 | 上交所/深交所/北交所/港交所/纳斯达克等 |
| 上市时间 | 首次公开发行并上市的日期 |
| 所属行业 | 证监会行业分类或申万行业分类 |
| 行业细分 | 所属三级或四级细分行业 |
| 注册地址 | 公司注册地址 |
| 办公地址 | 公司实际办公地址 |
| 成立日期 | 公司创立日期 |
| 法定代表人 | 公司董事长或执行董事 |
| 实际控制人 | 最终控制公司的自然人或机构 |
| 企业性质 | 国企/民企/外企/混合所有制 |

### 2.2 市值与股价数据

| 数据项 | 说明 |
|--------|------|
| 当前股价 | 最新收盘价 |
| 总市值 | 股价 × 总股本 |
| 流通市值 | 股价 × 流通股本 |
| 总股本 | 公司发行的全部股份 |
| 流通股本 | 可在二级市场交易的股份 |
| 限售股 | 暂时不能交易的股份 |

### 2.3 经营规模数据

| 数据项 | 说明 |
|--------|------|
| 员工人数 | 公司在职员工总数 |
| 研发人员 | 研发岗位员工数量 |
| 研发人员占比 | 研发人员÷员工总数×100% |
| 子公司数量 | 控股/参股子公司总数 |
| 分支机构 | 分公司、办事处数量 |
| 生产基地 | 生产工厂、基地数量 |
| 销售网络 | 销售网点、渠道覆盖 |

### 2.4 业务构成数据

| 数据项 | 说明 |
|--------|------|
| 分业务收入 | 各业务板块的营业收入 |
| 分业务收入占比 | 各业务收入÷总营收×100% |
| 分业务成本 | 各业务板块的营业成本 |
| 分业务毛利 | 各业务收入 - 各业务成本 |
| 分业务毛利率 | 各业务毛利÷各业务收入×100% |
| 分地区收入 | 各地区的营业收入 |
| 分地区收入占比 | 各地区收入÷总营收×100% |

### 2.5 联系方式与其他

| 数据项 | 说明 |
|--------|------|
| 公司官网 | 公司官方网站地址 |
| 投资者关系电话 | 投资者咨询热线 |
| 董秘邮箱 | 董事会秘书联系方式 |
| 信息披露媒体 | 指定信息披露报刊 |

### 2.6 数据期间要求

- **基础信息**：最新数据
- **业务构成**：最新年报或半年报数据
- **员工数据**：最新年报披露数据
- **数据优先级**：年报数据 > 半年报数据 > 季报数据

## 三、核心指标与计算方法

### 3.1 主营业务集中度指标

| 指标 | 计算公式 | 用途 |
|------|---------|------|
| 主营业务收入占比 | 主营业务收入÷总营收×100% | 衡量主业突出程度 |
| 最大业务占比 | 最大业务收入÷总营收×100% | 衡量单一业务依赖度 |
| 前三业务占比 | 前三大业务收入之和÷总营收×100% | 衡量业务集中度 |
| 业务数量 | 收入占比>5% 的业务数量 | 衡量业务多元化程度 |

### 3.2 业务多元化指标

| 指标 | 计算公式 | 用途 |
|------|---------|------|
| 业务集中度 CR3 | 前三大业务收入占比之和 | 衡量业务集中程度 |
| 业务集中度 CR5 | 前五大业务收入占比之和 | 衡量业务集中程度 |
| 多元化系数 | 1 - 最大业务占比 | 衡量多元化程度，越接近 1 越多元 |

### 3.3 人均效率指标

| 指标 | 计算公式 | 用途 |
|------|---------|------|
| 人均营收 | 营业收入÷员工人数 | 衡量人均产出效率 |
| 人均净利润 | 净利润÷员工人数 | 衡量人均盈利能力 |
| 人均市值 | 总市值÷员工人数 | 衡量人均市场价值 |

### 3.4 成长性业务识别指标

| 指标 | 计算方法 | 用途 |
|------|---------|------|
| 业务增速对比 | 各业务增速 vs 总营收增速 | 识别增长较快的业务 |
| 新兴业务收入占比 | 新兴业务收入÷总营收×100% | 衡量新业务发展程度 |
| 新兴业务增速 | 新兴业务本期增长额÷上期新兴业务收入×100% | 衡量新业务增长速度 |

## 四、分析逻辑与判断标准

### 4.1 主营业务集中度分析

**主营业务收入占比判断**：

| 主营业务收入占比 | 评估 | 说明 |
|-----------------|------|------|
| >70% | 高度集中 | 主业突出，专注度高 |
| 50-70% | 较集中 | 主业较为突出 |
| 30-50% | 较分散 | 业务较为多元 |
| <30% | 高度分散 | 业务高度多元化 |

**业务集中度与投资风格**：
- 主业集中：通常更容易建立竞争优势，但抗风险能力较弱
- 业务多元：抗风险能力强，但可能缺乏核心竞争力

### 4.2 业务多元化分析

**多元化类型判断**：

| 多元化类型 | 特征 | 评估 |
|-----------|------|------|
| 相关多元化 | 新业务与主业有协同 | 较好，可发挥协同效应 |
| 非相关多元化 | 新业务与主业无关联 | 较差，管理难度大 |
| 产业链延伸 | 向上游或下游延伸 | 较好，增强产业链控制 |
| 盲目跨界 | 追热点、无战略协同 | 差，风险较高 |

**多元化风险判断**：

| 情形 | 风险等级 | 说明 |
|------|---------|------|
| 主业稳健 + 相关多元化 | 低风险 | 协同发展 |
| 主业稳健 + 非相关多元化 | 中风险 | 管理挑战 |
| 主业下滑 + 跨界转型 | 高风险 | 可能两头不讨好 |
| 主业突出 | 低风险 | 专注优势领域 |

### 4.3 第二增长曲线分析

**第二增长曲线识别**：

| 特征 | 判断标准 |
|------|---------|
| 收入规模 | 新业务收入占比>10% 且<30% |
| 增长速度 | 新业务增速>30% 或显著快于主业 |
| 毛利率 | 新业务毛利率≥主业毛利率 |
| 战略地位 | 公司明确将新业务作为战略重点 |

**第二增长曲线发展阶段**：

| 阶段 | 特征 | 评估 |
|------|------|------|
| 培育期 | 收入占比<5%，高投入 | 风险高，需观察 |
| 成长期 | 收入占比 5-15%，高增长 | 潜力大，值得关注 |
| 成熟期 | 收入占比>15%，增速放缓 | 已形成规模 |
| 衰退期 | 负增长 | 第二曲线失效 |

### 4.4 人均效率分析

**人均营收判断**（因行业而异）：

| 行业类型 | 人均营收优秀标准 | 人均营收一般标准 |
|---------|----------------|----------------|
| 软件/互联网 | >100 万元/年 | 50-100 万元/年 |
| 金融 | >200 万元/年 | 100-200 万元/年 |
| 制造业 | >50 万元/年 | 20-50 万元/年 |
| 零售业 | >30 万元/年 | 10-30 万元/年 |
| 服务业 | >40 万元/年 | 15-40 万元/年 |

**人均效率趋势分析**：
- 人均营收上升：效率提升或产品升级
- 人均营收下降：效率下降或人员扩张过快
- 人均利润上升：盈利能力提升
- 人均利润下降：成本压力或竞争加剧

### 4.5 业务结构变化趋势分析

**业务结构变化分析**：

| 变化类型 | 特征 | 评估 |
|---------|------|------|
| 主业强化 | 主业占比提升 | 聚焦核心业务 |
| 主业弱化 | 主业占比下降 | 可能面临转型 |
| 新业务崛起 | 新业务占比快速提升 | 增长动力切换 |
| 业务收缩 | 多项业务占比下降 | 可能经营困难 |

**业务结构稳定性**：
- 业务结构稳定（各业务占比年变化<5%）：经营稳定
- 业务结构快速变化（年变化>10%）：战略调整或经营波动

### 4.6 行业差异化分析

**不同行业的业务结构特征**：

| 行业类型 | 典型业务结构 | 说明 |
|---------|-------------|------|
| 专业软件 | 单一主业>80% | 专注细分领域 |
| 消费电子 | 多元化 40-60% | 多产品线布局 |
| 家电 | 多元化 50-70% | 多品类发展 |
| 医药 | 核心产品 + 在研管线 | 产品迭代模式 |
| 互联网平台 | 核心业务 + 创新业务 | 生态化发展 |

## 五、综合评级标准

### 5.1 主业突出程度评级

| 评级 | 主营业务收入占比 | 说明 |
|------|----------------|------|
| 优秀 | >80% | 主业非常突出 |
| 良好 | 60-80% | 主业较为突出 |
| 一般 | 40-60% | 主业一般 |
| 偏弱 | 20-40% | 主业不突出 |
| 差 | <20% | 主业模糊 |

### 5.2 业务健康度评级

| 评级 | 业务结构 | 第二增长曲线 | 说明 |
|------|---------|-------------|------|
| 优秀 | 主业突出（>60%） | 明确且高增长 | 主业稳健 + 新业务发力 |
| 良好 | 主业突出（>60%） | 培育中 | 主业稳健，新业务待观察 |
| 一般 | 主业一般（40-60%） | 有 | 业务多元，有新增长点 |
| 偏弱 | 主业分散（20-40%） | 不明确 | 业务分散，方向不明 |
| 差 | 主业模糊（<20%） | 无 | 业务混乱，缺乏焦点 |

## 七、注意事项

1. **数据时效性**：业务构成数据通常来自年报/半年报，可能存在滞后
2. **业务分类变化**：公司可能调整业务分类口径，需注意数据可比性
3. **内部抵消**：分部收入之和可能与总营收存在差异（内部交易抵消）
4. **新兴业务识别**：新兴业务可能被归类在"其他业务"中，需仔细阅读年报
5. **地域分布**：出口型企业需关注各地区收入分布和贸易风险
6. **季节性因素**：部分业务存在季节性，需使用全年数据分析
7. **并购影响**：并购可能导致业务结构突然变化，需区分内生与外延
8. **信息完整性**：仅披露收入占比>10% 的业务是常见做法，小业务可能未披露
