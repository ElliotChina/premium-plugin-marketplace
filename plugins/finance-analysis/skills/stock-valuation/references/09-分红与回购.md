# 分红与回购分析

分红与回购分析旨在评估公司对股东的回报政策和回报能力，通过分红历史、分红率、股息率等指标判断公司是否重视股东回报，以及分红和回购的可持续性。这是价值投资分析中衡量公司股东文化的重要维度。

---

## 一、分析目标

1. **评估分红政策**：判断公司是否重视股东回报，分红政策是否稳定可持续
2. **分析分红能力**：通过盈利和现金流分析公司是否有能力持续分红
3. **计算股息回报**：计算股息率，与债券收益率对比判断配置价值
4. **分析回购动向**：通过回购分析公司对自身价值的判断和股东回报方式

---

## 二、需要采集的数据

### 2.1 分红历史数据

| 数据项 | 说明 |
|--------|------|
| 分红年度 | 分红所属年度 |
| 每股分红（税前） | 每股派发的现金红利（税前） |
| 每股分红（税后） | 个人投资者实际到手的红利 |
| 分红总额 | 当年分红总金额（亿元） |
| 分红比例 | 分红占净利润的比例 |
| 除权除息日 | 分红实施的日期 |
| 分红形式 | 现金分红/送股/转增 |

### 2.2 回购数据

| 数据项 | 说明 |
|--------|------|
| 回购期间 | 回购实施的时间段 |
| 回购数量 | 回购股份数量（股） |
| 回购金额 | 回购总金额（元） |
| 回购均价 | 平均回购价格 |
| 回购价格上限 | 回购方案规定的最高价 |
| 回购用途 | 注销/股权激励/员工持股 |
| 回购进度 | 已回购/实施中/已完成 |

### 2.3 其他相关数据

| 数据项 | 来源 | 用途 |
|--------|------|------|
| 净利润 | 利润表 | 计算分红率 |
| 经营活动现金流 | 现金流量表 | 分析分红能力 |
| 总股本 | 基本信息 | 计算每股数据 |
| 当前股价 | 行情数据 | 计算股息率 |
| 国债收益率 | 宏观数据 | 对比配置价值 |

### 2.4 数据期间要求

- **分红历史**：至少 5 年数据（用于分析分红稳定性和增长）
- **回购数据**：近 3 年数据
- **数据优先级**：公告数据 > 年报数据

## 三、核心指标与计算方法

### 3.1 分红指标

| 指标 | 计算公式 | 用途 |
|------|---------|------|
| 分红率 | 分红总额 ÷ 净利润 × 100% | 衡量分红占利润的比例 |
| 每股分红 | 分红总额 ÷ 总股本 | 衡量每股获得的分红 |
| 股息率 | 每股分红 ÷ 当前股价 × 100% | 衡量分红的收益率 |
| 分红收益率（5 年平均） | 5 年分红总额 ÷ 5 年净利润总额 × 100% | 衡量长期分红水平 |

### 3.2 分红增长指标

| 指标 | 计算公式 | 用途 |
|------|---------|------|
| 每股分红增长率 | (本期每股分红 - 上期每股分红) ÷ 上期每股分红 × 100% | 衡量分红增长速度 |
| 分红 3 年 CAGR | (期末每股分红÷期初每股分红)^(1/3) - 1 | 衡量中期分红复合增速 |
| 分红 5 年 CAGR | (期末每股分红÷期初每股分红)^(1/5) - 1 | 衡量长期分红复合增速 |

### 3.3 回购指标

| 指标 | 计算公式 | 用途 |
|------|---------|------|
| 回购比例 | 回购数量 ÷ 总股本 × 100% | 衡量回购力度 |
| 回购金额占比 | 回购金额 ÷ 净利润 × 100% | 衡量回购投入 |
| 回购注销比例 | 注销数量 ÷ 回购数量 × 100% | 衡量回购对股东的实际回报 |

## 四、分析逻辑与判断标准

### 4.1 分红率分析

**分红率判断**：

| 分红率 | 评估 | 说明 |
|--------|------|------|
| >50% | 高分红 | 分红慷慨，回报股东 |
| 30-50% | 中等分红 | 分红合理 |
| 10-30% | 低分红 | 分红偏低，可能处于扩张期 |
| <10% | 很少分红 | 很少分红，可能全部用于再投资 |
| 0% | 不分红 | 不分红，需分析原因 |

**行业差异参考**：
- 银行：通常 30% 左右（监管要求 + 成熟行业）
- 白酒：通常 40-60%（现金流好，资本开支少）
- 公用事业（电力、高速）：通常 40-60%（现金流稳定）
- 科技/成长股：通常<30%（需要资金投入研发和扩张）
- 房地产：波动较大（受政策和销售影响）

**分红率与生命周期**：
- **初创期**：不分红（需要资金扩张）
- **成长期**：低分红（需要资金再投资）
- **成熟期**：中高分红（现金流稳定，投资机会减少）
- **衰退期**：高分红（业务收缩，资金无处投资）

### 4.2 分红稳定性分析

**分红历史类型**：

| 类型 | 特征 | 评估 |
|------|------|------|
| 持续分红型 | 连续 5 年以上分红 | 优秀，股东文化好 |
| 分红增长型 | 分红金额持续增长 | 优秀，盈利增长 + 股东回报 |
| 稳定分红型 | 分红率稳定在合理水平 | 良好，分红政策稳定 |
| 波动分红型 | 分红金额波动大 | 一般，盈利波动大 |
| 偶尔分红型 | 不定期分红 | 较差，股东回报意识弱 |
| 铁公鸡型 | 多年不分红 | 差，无股东回报意识 |

**分红增长判断**：
- 连续 3 年以上分红增长：分红成长性好
- 分红增长率 > 利润增长率：分红意愿提升
- 分红增长率 < 利润增长率：分红意愿下降

### 4.3 股息率分析

**股息率判断**：

| 股息率 | 评估 | 说明 |
|--------|------|------|
| >5% | 高股息 | 股息回报丰厚 |
| 4-5% | 较高股息 | 股息回报较好 |
| 3-4% | 中等股息 | 股息回报一般 |
| 2-3% | 较低股息 | 股息回报较低 |
| <2% | 低股息 | 股息回报很低 |

**与国债收益率对比**：

| 对比 | 评估 | 说明 |
|------|------|------|
| 股息率>2 倍国债收益率 | 极具配置价值 | 股息回报远超债券 |
| 股息率>国债收益率 | 有配置价值 | 股息回报高于债券 |
| 股息率<国债收益率 | 配置价值一般 | 股息回报低于债券 |

**注**：10 年期国债收益率通常在 2.5-3.5% 之间

**行业差异参考**：
- 银行：股息率通常 4-6%（股价低，分红稳定）
- 公用事业：股息率通常 3-5%（防御性资产）
- 白酒：股息率通常 1-3%（股价高，成长性强）
- 科技股：股息率通常<1%（成长性强，分红少）

### 4.4 分红能力分析

**分红能力判断**：

| 指标 | 有能力分红 | 无能力分红 |
|------|-----------|-----------|
| 盈利情况 | 持续盈利 | 亏损或微利 |
| 经营现金流 | 持续为正且增长 | 持续为负或恶化 |
| 未分配利润 | 累积未分配利润高 | 未分配利润少或为负 |
| 资产负债率 | <60% | >70% |
| 现金覆盖债务 | >1 | <0.5 |

**分红能力与分红意愿匹配分析**：

| 分红能力 | 分红意愿（分红率） | 结论 |
|---------|------------------|------|
| 强 | 高 | 优秀，有能力且愿意分红 |
| 强 | 低 | 守财奴型，有钱不分 |
| 弱 | 高 | 勉强分红，不可持续 |
| 弱 | 低 | 正常，无能力分红 |

### 4.5 回购分析

**回购类型及意义**：

| 回购用途 | 对股东意义 | 评估 |
|---------|-----------|------|
| 回购注销 | 减少股本，提高每股收益 | 最优，直接回报股东 |
| 股权激励/员工持股 | 绑定员工利益 | 中等，可能摊薄股东 |
| 库存股 | 未来用于并购或激励 | 一般，用途不确定 |

**回购力度判断**：

| 回购比例 | 评估 | 说明 |
|---------|------|------|
| >3% | 大额回购 | 回购力度大，信心足 |
| 1-3% | 中等回购 | 回购力度适中 |
| 0.5-1% | 小额回购 | 回购力度较小 |
| <0.5% | 微量回购 | 象征性回购 |

**回购信号分析**：
- 低位回购：认为股价低估，信心足
- 高位回购：可能为完成股权激励任务
- 回购未完成：可能资金紧张或股价过高
- 回购超额完成：信心强烈

### 4.6 分红 + 回购综合回报

**综合回报率计算**：
```
综合回报率 = 股息率 + 回购收益率
回购收益率 = 回购金额 ÷ 市值 × 100%
```

**综合回报判断**：

| 综合回报率 | 评估 | 说明 |
|-----------|------|------|
| >5% | 高回报 | 股东回报丰厚 |
| 3-5% | 中等回报 | 股东回报合理 |
| 1-3% | 较低回报 | 股东回报较低 |
| <1% | 低回报 | 股东回报很低 |

## 七、注意事项

1. **除权除息**：分红后股价会除息，实际收益需考虑股价变化
2. **红利税**：个人投资者需缴纳红利税（持股 1 年以上免税）
3. **分红预案**：年报披露的为预案，需股东大会通过
4. **中期分红**：部分公司有中期分红，需合并计算
5. **特殊分红**：关注一次性特别分红（如资产处置后）
6. **回购进展**：关注回购方案是否实际执行
7. **回购注销**：只有回购注销才真正减少股本
8. **分红能力变化**：关注盈利下滑对分红能力的影响
