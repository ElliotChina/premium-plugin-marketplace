# 风险分析

风险分析旨在系统性地识别和评估公司面临的各种风险因素，包括行业风险、公司风险和财务风险三大维度。通过风险分析，可以全面了解公司可能面临的负面因素，为投资决策提供风险警示和规避建议。

---

## 一、分析目标

1. **识别行业风险**：分析政策变化、技术替代、行业周期等外部风险
2. **评估公司风险**：分析商誉减值、客户集中、管理层稳定性等内部风险
3. **排查财务风险**：分析高负债、现金流恶化、存货积压等财务隐患
4. **综合风险评级**：汇总各类风险，给出整体风险等级评估

---

## 二、需要采集的数据

### 2.1 行业风险数据

| 数据项 | 说明 |
|--------|------|
| 行业相关政策 | 影响行业的法律法规、监管政策、产业政策 |
| 政策变化趋势 | 政策是收紧还是放松，支持还是限制 |
| 技术迭代速度 | 行业技术更新换代的速度和趋势 |
| 颠覆性技术 | 可能颠覆现有商业模式的新兴技术 |
| 行业周期位置 | 行业处于上行周期还是下行周期 |
| 周期性波动幅度 | 行业周期波动的剧烈程度 |

### 2.2 公司风险数据

| 数据项 | 说明 |
|--------|------|
| 商誉金额 | 资产负债表中的商誉账面价值 |
| 商誉/净资产 | 商誉占净资产的比例 |
| 商誉来源 | 商誉形成的并购标的和时间 |
| 并购标的业绩承诺 | 并购时的业绩承诺和完成情况 |
| 前五客户收入占比 | 前五大客户贡献的收入比例 |
| 单一最大客户占比 | 最大客户贡献的收入比例 |
| 核心管理层信息 | 实控人、CEO、CFO 等核心高管 |
| 管理层变动记录 | 核心高管离职、接任情况 |
| 实控人股权质押 | 实控人股权质押比例和质押金额 |
| 管理层负面舆情 | 管理层相关的负面新闻或处罚 |

### 2.3 财务风险数据

| 数据项 | 说明 |
|--------|------|
| 资产负债率 | 总负债÷总资产×100% |
| 有息负债 | 短期借款 + 长期借款 + 应付债券 |
| 货币资金 | 资产负债表中的现金及等价物 |
| 经营活动现金流净额 | 现金流量表中的经营现金流 |
| 经营活动现金流趋势 | 近 3 年经营现金流变化趋势 |
| 存货账面价值 | 资产负债表中的存货金额 |
| 存货/营业收入 | 存货占营业收入的比例 |
| 存货周转率 | 营业成本÷平均存货 |
| 存货跌价准备 | 计提的存货跌价准备金额 |
| 应收账款账面价值 | 资产负债表中的应收账款金额 |
| 应收账款/营业收入 | 应收账款占营业收入的比例 |
| 应收账款周转率 | 营业收入÷平均应收账款 |
| 坏账准备 | 计提的应收账款坏账准备 |

### 2.4 数据期间要求

- **行业风险**：最新政策和技术动态
- **公司风险**：最新季报/年报数据
- **财务风险**：至少 3 年趋势数据
- **数据优先级**：年报数据 > 季报数据 > 业绩预告

## 三、核心指标与计算方法

### 3.1 行业风险指标

| 指标 | 计算方法 | 用途 |
|------|---------|------|
| 政策风险评分 | 根据政策变化方向和影响程度评分（1-5 分） | 衡量政策风险 |
| 技术替代风险评分 | 根据技术迭代速度和颠覆性技术威胁评分（1-5 分） | 衡量技术风险 |
| 周期风险评分 | 根据行业周期位置和波动幅度评分（1-5 分） | 衡量周期风险 |

### 3.2 商誉风险指标

| 指标 | 计算公式 | 用途 |
|------|---------|------|
| 商誉/净资产 | 商誉÷净资产×100% | 衡量商誉风险敞口 |
| 商誉/总资产 | 商誉÷总资产×100% | 衡量商誉资产占比 |
| 业绩承诺完成率 | 实际业绩÷承诺业绩×100% | 衡量并购标的是否达标 |

### 3.3 客户集中度指标

| 指标 | 计算公式 | 用途 |
|------|---------|------|
| 前五客户收入占比 | 前五客户收入÷总营收×100% | 衡量客户集中风险 |
| 单一最大客户占比 | 最大客户收入÷总营收×100% | 衡量单一客户依赖 |
| 最大客户占比/第二客户占比 | 衡量客户结构均衡性 |

### 3.4 管理层风险指标

| 指标 | 计算方法 | 用途 |
|------|---------|------|
| 核心高管变动率 | 变动高管数÷核心高管总数×100% | 衡量管理层稳定性 |
| 实控人质押率 | 质押股份÷实控人持股×100% | 衡量实控人资金压力 |
| 管理层负面舆情次数 | 统计负面舆情数量 | 衡量声誉风险 |

### 3.5 财务风险指标

| 指标 | 计算公式 | 用途 |
|------|---------|------|
| 资产负债率 | 总负债÷总资产×100% | 衡量负债水平 |
| 现金覆盖债务 | 货币资金÷有息负债 | 衡量偿债能力 |
| 现金流趋势评分 | 根据 3 年经营现金流变化评分（1-5 分） | 衡量现金流健康度 |
| 存货/营收 | 存货÷营业收入×100% | 衡量存货积压程度 |
| 存货周转率变化 | 本期周转率 - 上期周转率 | 衡量周转效率变化 |
| 应收账款/营收 | 应收账款÷营业收入×100% | 衡量回款风险 |
| 应收账款周转率变化 | 本期周转率 - 上期周转率 | 衡量回款效率变化 |

## 四、分析逻辑与判断标准

### 4.1 行业风险分析

**政策风险判断**：

| 政策变化 | 风险等级 | 说明 |
|---------|---------|------|
| 政策大力支持 | 低 | 政策利好行业发展 |
| 政策维持中性 | 中 | 政策无明显影响 |
| 政策收紧限制 | 高 | 政策对行业形成压制 |
| 政策重大利空 | 极高 | 政策可能导致行业衰退 |

**技术替代风险判断**：

| 技术迭代 | 风险等级 | 说明 |
|---------|---------|------|
| 技术稳定 | 低 | 行业技术成熟，迭代慢 |
| 缓慢迭代 | 中低 | 技术逐步升级，无颠覆风险 |
| 快速迭代 | 中高 | 技术更新快，需持续跟进 |
| 颠覆性威胁 | 高 | 存在技术颠覆风险 |

**行业周期风险判断**：

| 周期位置 | 风险等级 | 说明 |
|---------|---------|------|
| 上行初期 | 低 | 周期向上，景气度提升 |
| 上行中期 | 中低 | 周期持续向好 |
| 上行末期 | 中高 | 周期接近顶部 |
| 下行初期 | 高 | 周期转向，景气度下滑 |
| 下行中期 | 中高 | 周期底部徘徊 |
| 下行末期 | 中低 | 周期接近底部，可能反转 |

**行业差异化参考**：

| 行业类型 | 政策风险 | 技术风险 | 周期风险 |
|---------|---------|---------|---------|
| 医药（创新药） | 高（集采政策） | 高（研发失败风险） | 中 |
| 白酒 | 中低 | 低 | 低 |
| 半导体 | 中（地缘政治） | 高（技术迭代快） | 高（强周期） |
| 银行 | 中高（监管政策） | 低 | 中（经济周期） |
| 房地产 | 高（调控政策） | 低 | 高（强周期） |
| 软件/SaaS | 低 | 中高 | 低 |

### 4.2 商誉减值风险分析

**商誉/净资产判断**：

| 商誉/净资产 | 风险等级 | 说明 |
|------------|---------|------|
| <10% | 安全 | 商誉占比低，减值影响有限 |
| 10-20% | 正常 | 商誉占比适中 |
| 20-30% | 关注 | 商誉占比较高，需关注 |
| 30-50% | 较高风险 | 商誉占比高，减值风险较大 |
| >50% | 高风险 | 商誉占比过高，大幅减值可能导致亏损 |

**业绩承诺完成情况判断**：

| 完成率 | 评估 | 说明 |
|--------|------|------|
| >100% | 达标 | 完成业绩承诺 |
| 80-100% | 基本达标 | 接近承诺业绩 |
| 50-80% | 未达标 | 业绩承诺完成较差 |
| <50% | 严重未达标 | 业绩大幅低于承诺 |

**商誉减值迹象**：
- 并购标的业绩承诺未完成
- 并购标的所在行业景气度下滑
- 并购标的核心团队离职
- 公司主动计提商誉减值准备

### 4.3 客户集中度风险分析

**前五客户收入占比判断**：

| 占比 | 风险等级 | 说明 |
|------|---------|------|
| <20% | 低风险 | 客户分散，依赖度低 |
| 20-40% | 正常 | 客户结构合理 |
| 40-60% | 较高风险 | 客户集中度高 |
| >60% | 高风险 | 严重依赖大客户 |

**单一最大客户占比判断**：

| 占比 | 风险等级 | 说明 |
|------|---------|------|
| <10% | 低风险 | 单一客户影响小 |
| 10-20% | 正常 | 单一客户占比合理 |
| 20-30% | 较高风险 | 依赖单一客户 |
| >30% | 高风险 | 严重依赖单一客户 |

**客户流失影响分析**：
- 失去最大客户对收入的影响 = 最大客户收入占比
- 失去前三大客户对收入的影响 = 前三大客户收入占比之和
- 评估客户流失后是否容易找到替代客户

**行业差异化参考**：

| 行业类型 | 前五客户占比正常范围 | 说明 |
|---------|-------------------|------|
| ToC 消费品 | <20% | 客户分散 |
| 软件/SaaS | 20-40% | 可能依赖大客户 |
| 制造业（ToB） | 30-50% | 大客户制常见 |
| 军工/政府 | >50% | 客户高度集中 |

### 4.4 管理层风险分析

| 变动情况 | 风险等级 | 说明 |

### 4.6 风险叠加分析

**多重风险叠加**：
- 单一风险可能可控，但多重风险叠加可能引发连锁反应
- 例如：高负债 + 现金流恶化 + 存货积压 = 资金链断裂高风险

**风险传导路径**：
- 行业风险 → 公司收入下滑 → 财务风险暴露
- 公司风险（如大客户流失） → 收入下滑 → 财务风险暴露
- 财务风险 → 融资困难 → 经营困难 → 更多风险

### 5.1 行业风险评级

| 评级 | 政策风险 | 技术风险 | 周期风险 | 说明 |
|------|---------|---------|---------|------|

| 较高风险 | 高 | 中高 | 中高 | 行业风险较高 |
| 高风险 | 极高 | 高 | 高 | 行业风险很高 |

| 评级 | 商誉风险 | 客户集中风险 | 管理层风险 | 说明 |
|------|---------|-------------|-----------|------|
| 低风险 | 安全 | 低风险 | 低风险 | 公司内部稳定 |
| 中风险 | 正常 | 正常 | 正常 | 存在一定风险 |
| 较高风险 | 较高风险 | 较高风险 | 较高风险 | 风险较高 |
| 高风险 | 高风险 | 高风险 | 高风险 | 风险很高 |

### 5.3 财务风险评级

### 5.4 综合风险评级

| 评级 | 行业风险 | 公司风险 | 财务风险 | 说明 |
|------|---------|---------|---------|------|

## 七、注意事项

1. **风险动态变化**：风险不是一成不变的，需要持续跟踪和更新
2. **风险关联性**：注意风险之间的关联性和传导效应
3. **行业差异**：不同行业的风险标准不同，应与同行对比
4. **风险敞口**：区分风险存在和风险敞口大小（如商誉绝对值）
5. **先行指标**：关注风险的先行指标（如订单下滑先于收入下滑）
6. **黑天鹅事件**：警惕难以预测的黑天鹅事件
7. **风险定价**：高风险可能对应高收益，需综合评估风险收益比
8. **信息局限性**：基于公开信息分析，可能存在信息滞后或不完整
