# 现金流分析

现金流分析旨在评估公司现金的获取能力和使用效率，通过经营现金流、自由现金流等指标判断公司利润的真实性和可持续发展能力。现金流是企业的血液，比利润更能反映企业的真实经营状况。

---

## 一、分析目标

1. **验证利润质量**：通过现金流与利润的对比，判断利润是否真实可靠
2. **评估造血能力**：分析公司通过经营获取现金的能力
3. **判断分红能力**：通过自由现金流判断公司是否有能力分红和回购
4. **发现资金风险**：识别现金流断裂或资金链紧张的风险

---

### 2.2 其他相关数据

| 数据项 | 来源 | 用途 |
|--------|------|------|
| 营业收入 | 利润表 | 计算收入现金比 |
| 净利润 | 利润表 | 计算净现比 |
| 资本开支 | 投资活动现金流 | 计算自由现金流 |
| 折旧与摊销 | 现金流量表附注 | 分析企业非现金支出 |

### 2.3 数据期间要求

- **完整分析**：至少 3 年现金流量表数据（用于趋势分析）
- **三年净现比**：需要 3 年累计数据
- **数据优先级**：年报数据 > 季报数据

## 三、核心指标与计算方法

### 3.1 现金收入比指标

| 指标 | 计算公式 | 用途 |
|------|---------|------|
| 收入现金比 | 销售商品提供劳务收到的现金 ÷ 营业收入 × 100% | 衡量收入中现金的比例 |
| 净现比 | 经营活动现金流净额 ÷ 净利润 × 100% | 衡量净利润中现金的比例 |
| 三年净现比 | 3 年经营现金流合计 ÷ 3 年净利润合计 × 100% | 衡量长期利润的现金含量 |

### 3.2 自由现金流指标

| 指标 | 计算公式 | 用途 |
|------|---------|------|
| 自由现金流 (FCF) | 经营活动现金流净额 - 资本开支 | 企业真正可支配的现金 |
| FCF/净利润 | 自由现金流 ÷ 净利润 × 100% | 衡量自由现金流与利润的关系 |
| FCF/营收 | 自由现金流 ÷ 营业收入 × 100% | 衡量单位收入产生的自由现金 |

### 3.3 现金流结构指标

| 指标 | 计算公式 | 用途 |
|------|---------|------|
| 经营现金流占比 | 经营现金流净额 ÷ 现金及等价物净增加额 × 100% | 衡量现金增加的来源 |
| 资本开支率 | 资本开支 ÷ 经营活动现金流净额 × 100% | 衡量再投资强度 |
| 分红保障倍数 | 经营活动现金流净额 ÷ 现金分红 × 100% | 衡量现金流对分红的覆盖 |

## 四、分析逻辑与判断标准

### 4.1 收入现金比分析

**收入现金比判断**：

| 收入现金比 | 评估 | 说明 |
|-----------|------|------|
| >100% | 优秀 | 收入全部为现金，还有预收款 |
| 90-100% | 良好 | 收入基本为现金 |
| 80-90% | 正常 | 有少量应收账款 |
| 60-80% | 关注 | 应收账款较多 |
| <60% | 风险 | 收入大部分为应收账款 |

**行业差异参考**：
- 预收款/会员制（如白酒、软件）：通常>110%（有预收账款）
- 零售/快消：100-110%（现款现货）
- 制造业：90-100%（有一定账期）
- 工程/设备：70-90%（账期较长，分期收款）

**收入现金比趋势**：
- 上升：回款改善或预收款增加
- 下降：回款恶化或信用政策放宽

### 4.2 净现比分析

**净现比判断**：

| 净现比 | 评估 | 说明 |
|--------|------|------|
| >100% | 优秀 | 利润全部转化为现金 |
| 80-100% | 良好 | 利润大部分转化为现金 |
| 60-80% | 正常 | 利润部分转化为现金 |
| 40-60% | 关注 | 利润现金含量较低 |
| <40% | 风险 | 利润大部分未转化为现金 |
| <0% | 危险 | 盈利但现金流为负 |

**净现比与利润质量**：
- 净现比持续>100%：利润质量优秀，可能有预收款
- 净现比持续 80-100%：利润质量良好
- 净现比波动大：利润稳定性差
- 净现比持续<60%：可能存在盈余管理或利润虚增

**三年净现比判断**：

| 三年净现比 | 评估 | 说明 |
|-----------|------|------|
| >80% | 优秀 | 长期利润现金含量高 |
| 60-80% | 良好 | 长期利润现金含量较高 |
| 40-60% | 一般 | 长期利润现金含量中等 |
| <40% | 较差 | 长期利润现金含量低 |

### 4.3 自由现金流分析

**自由现金流判断**：

| FCF | 评估 | 说明 |
|-----|------|------|
| 持续为正且增长 | 优秀 | 造血能力强，可分红回购 |
| 持续为正但稳定 | 良好 | 造血能力稳定 |
| 波动（有正有负） | 一般 | 受投资周期影响 |
| 持续为负 | 风险 | 需要外部融资维持 |

**自由现金流与生命周期**：
- **成长期公司**：FCF 可能为负（大量资本开支扩张）
- **成熟期公司**：FCF 应为正且稳定
- **衰退期公司**：FCF 可能为正但营收下滑（缩减投资）

**自由现金流用途分析**：
- **分红**：FCF 应>现金分红
- **回购**：FCF 充足时可持续回购
- **再投资**：FCF 用于扩张需关注投资回报率
- **还债**：FCF 用于还债降低杠杆

### 4.4 现金流结构分析

**现金流类型组合分析**：

| 经营现金流 | 投资现金流 | 筹资现金流 | 类型 | 说明 |
|-----------|-----------|-----------|------|------|
| + | - | - | 成熟型 | 经营造血，投资扩张，偿还债务 |
| + | - | + | 扩张型 | 经营造血，投资扩张，同时融资 |
| + | + | - | 收缩型 | 经营造血，变卖资产，偿还债务 |
| - | - | + | 初创型 | 经营未造血，投资扩张，依赖融资 |
| - | + | + | 困境型 | 经营未造血，变卖资产，依赖融资 |
| + | + | + | 特殊型 | 经营造血，变卖资产，同时融资 |

**注**：+ 表示净流入，- 表示净流出

**理想现金流结构**：
- 经营现金流：应为正（造血能力）
- 投资现金流：可为负（持续投资）
- 筹资现金流：成熟公司应为负（还债或分红）

### 4.5 现金流趋势分析

**经营现金流趋势**：
- 持续增长：经营能力增强
- 稳定：经营能力稳定
- 持续下降：经营能力恶化
- 波动大：经营不稳定

**经营现金流与营收增长匹配性**：
- 经营现金流增速 ≈ 营收增速：健康增长
- 经营现金流增速 > 营收增速：回款改善或预收款增加
- 经营现金流增速 < 营收增速：回款恶化或应收账款增加

### 4.6 分红保障分析

**分红保障倍数判断**：

| 分红保障倍数 | 评估 | 说明 |
|-------------|------|------|
| >2 | 优秀 | 现金流对分红覆盖度高 |
| 1.5-2 | 良好 | 现金流可覆盖分红 |
| 1-1.5 | 正常 | 现金流勉强覆盖分红 |
| <1 | 风险 | 现金流无法覆盖分红 |

## 七、注意事项

1. **季节性因素**：部分行业现金流存在明显季节性，应使用全年数据
2. **会计政策**：关注应收票据是否计入销售商品收到现金
3. **资本开支定义**：通常为"购建固定资产、无形资产等支付的现金"
4. **非现金项目**：折旧摊销是非现金支出，应在分析中加回
5. **营运资本变动**：应收账款、存货增加会占用现金流
6. **表外因素**：关注票据贴现、保理等对现金流的影响
7. **一次性影响**：剔除大额非经常性现金收支的影响
8. **集团资金归集**：部分公司现金流可能被集团归集，需关注其他应收款
