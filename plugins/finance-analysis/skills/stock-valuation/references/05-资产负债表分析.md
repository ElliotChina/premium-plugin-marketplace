# 资产负债表分析

资产负债表分析旨在评估公司的财务安全性和资产质量，通过偿债能力、资产结构、营运效率等维度判断公司是否存在财务风险，资产是否健康有效。

---

## 一、分析目标

1. **评估偿债能力**：判断公司短期和长期债务的偿还能力
2. **分析资产质量**：识别资产是否存在减值、虚高或低效问题
3. **判断财务安全**：评估现金对债务的覆盖程度
4. **发现潜在风险**：识别商誉、存货、应收账款等风险点

---

## 二、需要采集的数据

### 2.1 资产负债表核心数据

| 数据项 | 说明 |
|--------|------|
| **资产端** | |
| 流动资产合计 | 一年内可变现的资产 |
| 货币资金 | 现金及银行存款 |
| 应收账款 | 客户欠款 |
| 存货 | 原材料、在产品、产成品 |
| 非流动资产合计 | 长期资产 |
| 固定资产 | 厂房、设备等 |
| 在建工程 | 未完工的工程 |
| 无形资产 | 专利、商标、土地使用权 |
| 商誉 | 并购形成的溢价 |
| 资产总计 | 总资产 |
| **负债端** | |
| 流动负债合计 | 一年内到期的债务 |
| 短期借款 | 一年内需偿还的借款 |
| 应付账款 | 欠供应商的款项 |
| 非流动负债合计 | 长期债务 |
| 长期借款 | 一年后到期的借款 |
| 应付债券 | 发行的债券 |
| 负债合计 | 总负债 |
| **权益端** | |
| 实收资本/股本 | 股东投入资本 |
| 资本公积 | 资本溢价等 |

| 归属于母公司股东权益合计 | 净资产 |

### 2.2 数据期间要求

- **完整分析**：至少 3 年资产负债表数据（用于趋势分析）
- **时点数据**：资产负债表为时点数据，关注期末数
- **数据优先级**：年报数据 > 季报数据

## 三、核心指标与计算方法

### 3.1 偿债能力指标

| 指标 | 计算公式 | 用途 |
|------|---------|------|
| 资产负债率 | 总负债 ÷ 总资产 × 100% | 衡量整体负债水平 |
| 流动比率 | 流动资产 ÷ 流动负债 | 衡量短期偿债能力 |
| 速动比率 | (流动资产 - 存货) ÷ 流动负债 | 更严格的短期偿债能力 |
| 现金比率 | 货币资金 ÷ 流动负债 | 最保守的偿债能力 |

### 3.2 现金安全指标

| 指标 | 计算公式 | 用途 |
|------|---------|------|
| 现金覆盖债务 | 货币资金 ÷ 有息负债 | 衡量现金对债务的覆盖 |
| 有息负债率 | 有息负债 ÷ 总资产 × 100% | 衡量有息债务负担 |

**有息负债** = 短期借款 + 一年内到期的非流动负债 + 长期借款 + 应付债券

### 3.3 资产质量指标

| 指标 | 计算公式 | 用途 |
|------|---------|------|
| 商誉/净资产 | 商誉 ÷ 净资产 × 100% | 衡量商誉减值风险 |
| 存货/营收 | 存货 ÷ 营业收入 × 100% | 衡量存货积压风险 |
| 应收账款/营收 | 应收账款 ÷ 营业收入 × 100% | 衡量回款风险 |

### 3.4 营运效率指标

| 指标 | 计算公式 | 用途 |
|------|---------|------|
| 存货周转率 | 营业成本 ÷ 平均存货 | 衡量存货周转速度 |
| 应收账款周转率 | 营业收入 ÷ 平均应收账款 | 衡量回款速度 |
| 总资产周转率 | 营业收入 ÷ 平均总资产 | 衡量资产使用效率 |

**平均存货** = (期初存货 + 期末存货) ÷ 2
**平均应收账款** = (期初应收账款 + 期末应收账款) ÷ 2
**平均总资产** = (期初总资产 + 期末总资产) ÷ 2

## 四、分析逻辑与判断标准

### 4.1 偿债能力分析

**资产负债率判断**：

| 资产负债率 | 评估 | 说明 |
|-----------|------|------|
| <40% | 安全 | 负债水平低，财务稳健 |
| 40-60% | 正常 | 合理负债水平 |
| 60-70% | 关注 | 负债偏高，需关注现金流 |
| >70% | 风险 | 高负债经营，风险较高 |

**行业差异参考**：
- 房地产：70-80% 属正常（高杠杆行业）
- 银行：85-90% 属正常（特殊经营模式）
- 制造业：50-60% 属正常
- 软件/互联网：30-40% 属正常（轻资产）

**流动比率判断**：

| 流动比率 | 评估 | 说明 |
|---------|------|------|
| >2 | 良好 | 短期偿债能力强 |
| 1.5-2 | 正常 | 短期偿债能力尚可 |
| 1-1.5 | 偏弱 | 短期偿债能力一般 |
| <1 | 风险 | 短期偿债能力不足 |

**速动比率判断**：

| 速动比率 | 评估 | 说明 |
|---------|------|------|
| >1.5 | 良好 | 快速偿债能力强 |
| 1-1.5 | 正常 | 快速偿债能力尚可 |
| 0.5-1 | 偏弱 | 快速偿债能力一般 |
| <0.5 | 风险 | 快速偿债能力不足 |

### 4.2 现金安全分析

**现金覆盖债务判断**：

| 现金覆盖债务 | 评估 | 说明 |
|-------------|------|------|
| >1 | 安全 | 现金足以覆盖全部有息债务 |
| 0.5-1 | 关注 | 现金覆盖一半左右债务 |
| 0.3-0.5 | 偏紧 | 现金覆盖能力较弱 |
| <0.3 | 高风险 | 现金严重不足，存在偿债风险 |

### 4.3 商誉风险分析

**商誉/净资产判断**：

| 商誉/净资产 | 风险等级 | 说明 |
|------------|---------|------|
| <10% | 安全 | 商誉占比低，减值风险小 |
| 10-30% | 正常 | 商誉占比适中 |
| 30-50% | 关注 | 商誉占比较高，需关注减值 |
| >50% | 高风险 | 商誉占比过高，减值可能重创业绩 |

**商誉减值测试要点**：
- 关注被收购资产业绩承诺是否完成
- 关注商誉形成时间（3 年以上需重点关注）
- 关注收购标的所属行业是否景气

### 4.4 存货风险分析

**存货/营收判断**：

| 存货/营收 | 评估 | 说明 |
|----------|------|------|
| <20% | 良好 | 存货周转快，积压风险小 |
| 20-40% | 正常 | 存货水平合理 |
| 40-60% | 关注 | 存货偏高，需关注周转 |
| >60% | 风险 | 存货积压严重 |

**存货周转率判断**：
- 周转率上升：存货周转加快，营运效率提升
- 周转率下降：存货周转放缓，可能存在积压

**行业差异参考**：
- 白酒行业：存货周转率 0.3-0.5 次/年属正常（陈酿需要时间）
- 快消品：存货周转率 5-10 次/年属正常
- 电子产品：存货周转率 4-8 次/年属正常（跌价风险高）
- 房地产：存货周转率 0.3-0.5 次/年属正常

**存货跌价风险判断**：
- 电子产品、时尚服饰：跌价风险高
- 食品、药品：有保质期，跌价风险中等
- 金属、原材料：受大宗商品价格波动影响

### 4.5 应收账款风险分析

**应收账款/营收判断**：

| 应收账款/营收 | 评估 | 说明 |
|--------------|------|------|
| <20% | 良好 | 回款快，资金占用少 |
| 20-40% | 正常 | 回款水平合理 |
| 40-60% | 关注 | 回款偏慢，资金占用多 |
| >60% | 风险 | 回款困难，坏账风险高 |

**应收账款周转率判断**：
- 周转率上升：回款速度加快
- 周转率下降：回款速度放缓，需关注坏账风险

**坏账风险判断**：
- 关注应收账款账龄结构（1 年以上占比）
- 关注坏账准备计提比例是否充分
- 关注前五大客户欠款占比是否过高

### 4.6 营运效率分析

**总资产周转率判断**：

| 总资产周转率 | 评估 | 说明 |
|-------------|------|------|
| >1 | 优秀 | 资产使用效率高 |
| 0.5-1 | 正常 | 资产使用效率尚可 |
| 0.3-0.5 | 偏低 | 资产使用效率一般 |
| <0.3 | 低效 | 资产使用效率低 |

**行业差异参考**：
- 零售/贸易：总资产周转率 2-4 次/年属正常（薄利多销）
- 制造业：总资产周转率 0.5-1.5 次/年属正常
- 重资产行业：总资产周转率 0.3-0.8 次/年属正常

## 七、注意事项

1. **行业特性**：不同行业资产负债结构差异大，应与同行业对比
2. **季节性**：部分行业资产负债表存在季节性波动
3. **会计政策**：关注存货计价方法、坏账计提政策等会计估计
4. **表外负债**：关注担保、诉讼等或有负债
5. **资产受限**：关注货币资金是否存在使用限制（如保证金）
6. **关联方占用**：关注其他应收款是否存在关联方资金占用
7. **有息负债识别**：准确区分有息负债和无息负债（应付账款等）
8. **商誉减值**：关注收购资产业绩承诺完成情况和商誉减值测试
