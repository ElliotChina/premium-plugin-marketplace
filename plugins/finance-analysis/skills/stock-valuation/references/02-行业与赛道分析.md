# 行业与赛道分析

行业与赛道分析旨在全面了解公司所处行业的发展空间、竞争格局、进入壁垒和增长驱动因素。行业是公司发展的大环境，好行业才能孕育好公司。通过行业分析，可以判断公司所处赛道是否足够宽广、是否具备长期投资价值。

---

## 一、分析目标

1. **评估行业空间**：分析行业市场规模和天花板
2. **判断行业生命周期**：确定行业处于导入期、成长期、成熟期还是衰退期
3. **分析行业格局**：了解行业集中度和竞争态势
4. **识别行业壁垒**：评估行业进入门槛和护城河
5. **挖掘驱动因素**：找出行业增长的核心驱动力

---

## 二、需要采集的数据

### 2.1 行业规模数据

| 数据项 | 说明 |
|--------|------|
| 行业市场规模 | 行业整体市场规模（亿元） |
| 市场规模历史数据 | 过去 5-10 年市场规模变化 |
| 市场规模预测 | 未来 3-5 年市场规模预测 |
| 行业增长率 | 行业同比增速 |
| 行业 5 年 CAGR | 行业 5 年复合增长率 |
| 全球市场规模 | 全球范围内行业规模（如有） |
| 中国市场规模占比 | 中国市场占全球比例 |

### 2.2 行业生命周期数据

| 数据项 | 说明 |
|--------|------|
| 行业渗透率 | 产品/服务在目标人群中的渗透比例 |
| 行业产能利用率 | 行业整体产能利用情况 |
| 企业数量变化 | 行业内企业数量增减 |
| 新进入者数量 | 新进入行业的企业数量 |
| 退出企业数量 | 退出行业的企业数量 |
| 行业投资热度 | VC/PE 投资金额和案例数 |

### 2.3 行业竞争格局数据

| 数据项 | 说明 |
|--------|------|
| 行业龙头企业 | 市场份额前几名的公司 |
| 各企业市场份额 | 各公司市占率 |
| CR3（行业前三集中度） | 前三大企业市场份额之和 |
| CR5（行业前五集中度） | 前五大企业市场份额之和 |
| CR10（行业前十集中度） | 前十大企业市场份额之和 |
| 赫芬达尔指数 HHI | 衡量市场集中度的指标 |
| 行业价格战情况 | 是否存在价格竞争 |

### 2.4 行业壁垒数据

| 壁垒类型 | 需要采集的数据 |
|---------|---------------|
| 技术壁垒 | 专利技术数量、研发投入占比、技术人才储备 |
| 资金壁垒 | 行业初始投资规模、资金密集程度 |
| 牌照壁垒 | 是否需要特殊资质或许可证 |
| 品牌壁垒 | 品牌认知度、品牌忠诚度、品牌建设周期 |
| 渠道壁垒 | 渠道建设难度、渠道覆盖要求 |
| 规模壁垒 | 规模经济效应、最小经济规模 |
| 转换成本 | 客户更换供应商的成本和难度 |
| 政策壁垒 | 行业准入政策、外资限制等 |

### 2.5 行业驱动因素数据

| 驱动类型 | 需要采集的数据 |
|---------|---------------|
| 人口驱动 | 目标人群规模、人口结构变化、城镇化率 |
| 技术驱动 | 技术创新速度、新技术应用、技术迭代周期 |
| 政策驱动 | 产业政策、补贴政策、监管政策 |
| 消费升级 | 人均收入增长、消费结构变化、品质升级 |
| 出口驱动 | 出口占比、海外市场增速、贸易环境 |
| 产业链驱动 | 上下游发展、产业转移、国产替代 |

### 2.6 行业风险数据

| 数据项 | 说明 |
|--------|------|
| 政策风险 | 政策变化方向和影响 |
| 技术替代风险 | 颠覆性技术威胁 |
| 周期波动风险 | 行业周期性波动幅度 |
| 原材料价格风险 | 主要原材料价格波动 |
| 汇率风险 | 汇率波动对行业的影响 |
| 国际贸易风险 | 贸易摩擦、关税影响 |

### 2.7 数据期间要求

- **行业规模**：至少 5 年历史数据 + 未来 3-5 年预测
- **竞争格局**：最新市场份额数据
- **渗透率**：最新数据 + 历史趋势
- **数据优先级**：权威机构报告 > 行业协会数据 > 上市公司年报

## 三、核心指标与计算方法

### 3.1 行业空间指标

| 指标 | 计算公式 | 用途 |
|------|---------|------|
| 行业市场规模 | 行业总销售额或总产值 | 衡量行业体量 |
| 行业增长率 | （本期规模 - 上期规模）÷上期规模×100% | 衡量行业增速 |
| 行业 5 年 CAGR | （期末规模÷期初规模）^（1/5）- 1 | 衡量中期复合增速 |
| 人均消费额 | 行业规模÷目标人群数量 | 衡量人均消费水平 |
| 渗透率 | 已有用户数÷潜在用户总数×100% | 衡量市场开发程度 |

### 3.2 行业集中度指标

| 指标 | 计算公式 | 用途 |
|------|---------|------|
| CR3 | 前三大企业市占率之和 | 衡量行业集中度 |
| CR5 | 前五大企业市占率之和 | 衡量行业集中度 |
| CR10 | 前十大企业市占率之和 | 衡量行业集中度 |
| HHI 指数 | 各企业市占率平方和 | 更精确的集中度指标 |

### 3.3 行业生命周期指标

| 指标 | 计算方法 | 用途 |
|------|---------|------|
| 渗透率 | 已有用户÷潜在用户×100% | 判断生命周期阶段 |
| 行业增速 vs GDP 增速 | 行业增速÷GDP 增速 | 判断行业相对景气度 |
| 企业数量变化率 | （本期企业数 - 上期企业数）÷上期企业数×100% | 判断行业吸引力 |
| 产能利用率 | 实际产量÷设计产能×100% | 判断供需关系 |

### 3.4 行业壁垒评分指标

| 指标 | 评分方法 | 用途 |
|------|---------|------|
| 技术壁垒评分 | 1-5 分，根据技术难度和专利情况 | 衡量技术门槛 |
| 资金壁垒评分 | 1-5 分，根据初始投资规模 | 衡量资金门槛 |
| 牌照壁垒评分 | 1-5 分，根据资质获取难度 | 衡量政策门槛 |
| 品牌壁垒评分 | 1-5 分，根据品牌建设周期 | 衡量品牌门槛 |
| 综合壁垒评分 | 各项评分平均值 | 衡量整体进入难度 |

## 四、分析逻辑与判断标准

### 4.1 行业空间分析

**行业市场规模判断**：

| 行业规模 | 评估 | 说明 |
|---------|------|------|
| >1 万亿 | 超大市场 | 市场空间巨大 |
| 5000 亿 -1 万亿 | 大市场 | 市场空间较大 |
| 1000 亿 -5000 亿 | 中等市场 | 市场空间适中 |
| 500 亿 -1000 亿 | 小市场 | 市场空间有限 |
| <500 亿 | 微小市场 | 市场空间很小 |

**行业增长率判断**：

| 行业增速 | 评估 | 说明 |
|---------|------|------|
| >20% | 高增长 | 行业快速发展 |
| 15-20% | 较快增长 | 行业增长较快 |
| 10-15% | 中速增长 | 行业稳定增长 |
| 5-10% | 低速增长 | 行业增长放缓 |
| <5% | 停滞 | 行业基本停滞 |
| 负增长 | 衰退 | 行业萎缩 |

**渗透率与生命周期判断**：

| 渗透率 | 生命周期阶段 | 评估 |
|--------|-------------|------|
| <10% | 导入期 | 早期阶段，风险高 |
| 10-40% | 成长期早期 | 快速发展，机会大 |
| 40-70% | 成长期晚期 | 增速放缓，竞争加剧 |
| 70-90% | 成熟期 | 市场饱和，存量竞争 |
| >90% | 衰退期 | 市场萎缩，寻找新方向 |

### 4.2 行业竞争格局分析

**市场集中度判断**：

| CR5 | 市场类型 | 评估 |
|-----|---------|------|
| >70% | 高集中度市场 | 寡头垄断，格局稳定 |
| 50-70% | 中高度集中 | 龙头优势明显 |
| 30-50% | 中度集中 | 竞争较为激烈 |
| <30% | 分散市场 | 竞争激烈，格局未定 |

**HHI 指数判断**：

| HHI 指数 | 市场类型 | 说明 |
|---------|---------|------|
| >2500 | 高度集中市场 | 寡头垄断 |
| 1500-2500 | 中度集中市场 | 垄断竞争 |
| <1500 | 分散市场 | 充分竞争 |

**竞争格局演变趋势**：
- CR5 上升：行业整合，龙头优势增强
- CR5 下降：新进入者增多，竞争加剧
- CR5 稳定：格局稳定，强者恒强

**价格战判断**：
- 存在价格战：行业盈利能力下降，加速出清
- 无价格战：竞争相对理性，盈利稳定

### 4.3 行业壁垒分析

**技术壁垒判断**：

| 技术壁垒程度 | 特征 | 评估 |
|-------------|------|------|
| 高 | 专利密集、研发投入>10%、技术迭代快 | 护城河深 |
| 中 | 有一定专利、研发 5-10%、技术稳步发展 | 有一定护城河 |
| 低 | 专利少、研发<5%、技术成熟稳定 | 护城河浅 |

**资金壁垒判断**：

| 资金壁垒程度 | 初始投资规模 | 评估 |
|-------------|-------------|------|
| 高 | >10 亿元 | 资金门槛高 |
| 中 | 1-10 亿元 | 有一定资金门槛 |
| 低 | <1 亿元 | 资金门槛低 |

**牌照壁垒判断**：

| 牌照壁垒程度 | 特征 | 评估 |
|-------------|------|------|
| 高 | 牌照稀缺、审批严格、外资禁入 | 政策护城河深 |
| 中 | 需要资质、但可获得 | 有一定政策门槛 |
| 低 | 无需特殊资质 | 无政策壁垒 |

**品牌壁垒判断**：

| 品牌壁垒程度 | 品牌建设周期 | 评估 |
|-------------|-------------|------|
| 高 | >10 年 | 品牌护城河深 |
| 中 | 5-10 年 | 有一定品牌壁垒 |
| 低 | <5 年 | 品牌壁垒浅 |

**综合壁垒评估**：

| 综合壁垒评分 | 评估 | 说明 |
|-------------|------|------|
| 4-5 分 | 高壁垒 | 行业护城河深，新进入者难 |
| 3-4 分 | 中壁垒 | 有一定门槛 |
| 1-3 分 | 低壁垒 | 门槛低，竞争激烈 |

### 4.4 行业驱动因素分析

**人口驱动判断**：

| 驱动因素 | 正面信号 | 负面信号 |
|---------|---------|---------|
| 人口规模 | 目标人群增长 | 人口红利消失 |
| 城镇化率 | 城镇化率提升 | 城镇化见顶 |
| 老龄化 | 银发经济机会 | 劳动力减少 |
| 家庭结构 | 小家庭增多 | - |

**技术驱动判断**：

| 驱动因素 | 正面信号 | 负面信号 |
|---------|---------|---------|
| 技术创新 | 新技术突破、专利增多 | 技术停滞 |
| 技术渗透 | 新技术快速普及 | 技术被替代 |
| 技术迭代 | 产品持续升级 | 迭代放缓 |

**政策驱动判断**：

| 驱动因素 | 正面信号 | 负面信号 |
|---------|---------|---------|
| 产业政策 | 政策扶持、税收优惠 | 政策收紧、限制发展 |
| 补贴政策 | 补贴持续 | 补贴退坡 |
| 监管政策 | 监管规范、利好龙头 | 监管趋严、成本增加 |

**消费升级判断**：

| 驱动因素 | 正面信号 | 负面信号 |
|---------|---------|---------|
| 人均收入 | 收入增长 | 收入下滑 |
| 消费结构 | 高端化、品质化 | 消费降级 |
| 品牌意识 | 品牌偏好增强 | 价格敏感 |

**出口驱动判断**：

| 驱动因素 | 正面信号 | 负面信号 |
|---------|---------|---------|
| 海外需求 | 海外需求增长 | 海外需求萎缩 |
| 贸易环境 | 贸易关系稳定 | 贸易摩擦加剧 |
| 汇率 | 汇率有利 | 汇率不利 |

### 4.5 行业生命周期综合判断

**各生命周期特征**：

| 特征 | 导入期 | 成长期 | 成熟期 | 衰退期 |
|------|-------|-------|-------|-------|
| 市场规模 | 小 | 快速扩大 | 达到峰值 | 萎缩 |
| 增长率 | 不稳定 | 20%+ | 5-10% | 负增长 |
| 渗透率 | <10% | 10-70% | 70-90% | >90% |
| 企业数量 | 少 | 快速增多 | 稳定或减少 | 减少 |
| 竞争程度 | 低 | 加剧 | 激烈 | 缓和 |
| 盈利能力 | 低或亏损 | 提升 | 稳定 | 下滑 |
| 技术变革 | 快速 | 较快 | 缓慢 | 停滞 |
| 投资风险 | 高 | 中 | 低 | 高 |

### 4.6 行业差异化分析

**不同行业的特征对比**：

| 行业类型 | 典型规模 | 典型增速 | 典型 CR5 | 典型壁垒 |
|---------|---------|---------|---------|---------|
| 互联网平台 | 万亿级 | 15-30% | >70% | 网络效应、规模 |
| 白酒 | 千亿级 | 10-15% | >50% | 品牌、渠道 |
| 医药 | 万亿级 | 10-20% | 20-40% | 技术、牌照 |
| 半导体 | 万亿级 | 15-25% | >60% | 技术、资金 |
| 制造业 | 千亿级 | 5-10% | 30-50% | 规模、技术 |
| 零售 | 万亿级 | 5-15% | 20-40% | 渠道、规模 |

## 五、综合评级标准

### 5.1 行业吸引力评级

| 评级 | 行业空间 | 行业增速 | 进入壁垒 | 竞争格局 | 说明 |
|------|---------|---------|---------|---------|------|
| 优秀 | 大 | 高 | 高 | 集中 | 好行业，值得投资 |
| 良好 | 较大 | 较快 | 中 | 较集中 | 较好行业 |
| 一般 | 中等 | 中速 | 中 | 一般 | 普通行业 |
| 偏弱 | 较小 | 低速 | 低 | 分散 | 较差行业 |
| 差 | 小 | 负增长 | 低 | 分散 | 差行业，回避 |

### 5.2 行业生命周期评级

| 评级 | 生命周期阶段 | 说明 |
|------|-------------|------|
| 优秀 | 成长期早期 | 渗透率 10-40%，高速增长 |
| 良好 | 成长期中期 | 渗透率 40-60%，持续增长 |
| 一般 | 成长期晚期/成熟期 | 渗透率 60-90%，稳定增长 |
| 偏弱 | 成熟期晚期 | 渗透率>90%，增长停滞 |
| 差 | 衰退期 | 负增长，行业萎缩 |

## 七、注意事项

1. **数据来源可靠性**：优先采用权威机构、行业协会、政府部门数据
2. **口径一致性**：不同机构统计口径可能不同，需注意数据可比性
3. **动态变化**：行业格局和趋势会变化，需定期更新分析
4. **细分行业差异**：同一行业不同细分领域可能有很大差异
5. **周期因素**：周期行业需区分周期波动和长期趋势
6. **政策敏感性**：政策驱动型行业需密切关注政策变化
7. **技术颠覆风险**：技术密集型行业需警惕颠覆性技术
8. **全球化视角**：部分行业需从全球视角分析，而非仅看国内市场
