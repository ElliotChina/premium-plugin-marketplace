# 同行对比分析

同行对比分析旨在通过将目标公司与同行业可比公司进行多维度对比，评估公司在行业中的竞争地位、估值水平和综合实力。通过横向对比，可以更清晰地判断公司是行业龙头还是跟随者，估值是相对便宜还是昂贵。

---

## 一、分析目标

1. **确定行业地位**：判断公司是行业龙头、挑战者还是跟随者
2. **估值横向对比**：对比同行估值，判断相对估值水平
3. **财务指标对比**：对比盈利能力、成长性、财务健康度
4. **综合竞争力评分**：量化评估公司相对同行的竞争优势

---

## 二、需要采集的数据

### 2.1 同行业公司列表

| 数据项 | 说明 |
|--------|------|
| 同行业公司名称 | 与目标公司处于同一行业的上市公司 |
| 股票代码 | 同行公司代码 |
| 所属行业细分 | 确保细分行业一致（如白酒、化学制药、消费电子等） |
| 主营业务 | 确认业务可比性 |

### 2.2 基础数据

| 数据项 | 说明 |
|--------|------|
| 市值 | 各公司总市值（亿元） |
| 股价 | 当前股价 |
| 总股本 | 各公司总股本 |

### 2.3 估值数据

| 数据项 | 说明 |
|--------|------|
| PE(TTM) | 滚动市盈率 |
| PB | 市净率 |
| PS(TTM) | 滚动市销率 |
| PEG | 市盈率相对盈利增长比率 |
| 股息率 | 分红收益率 |

### 2.4 财务数据

| 数据项 | 说明 |
|--------|------|
| 营业收入（TTM） | 滚动 12 个月营收 |
| 营收增长率 | 营收同比增速 |
| 净利润（TTM） | 滚动 12 个月净利润 |
| 利润增长率 | 净利润同比增速 |
| 毛利率 | 毛利率 |
| 净利率 | 净利率 |
| ROE | 净资产收益率 |
| 资产负债率 | 负债率 |

### 2.5 数据期间要求

- **估值数据**：最新数据
- **财务数据**：TTM 或最新年报数据
- **同行数量**：至少 5 家可比公司（越多越有参考意义）

## 三、核心指标与计算方法

### 3.1 规模指标

| 指标 | 计算公式 | 用途 |
|------|---------|------|
| 市值排名 | 按市值从大到小排序 | 衡量行业地位 |
| 营收排名 | 按营收从大到小排序 | 衡量业务规模 |
| 利润排名 | 按净利润从大到小排序 | 衡量盈利规模 |

### 3.2 估值相对指标

| 指标 | 计算公式 | 用途 |
|------|---------|------|
| 相对 PE | 公司 PE ÷ 行业平均 PE | 衡量相对估值水平 |
| 相对 PB | 公司 PB ÷ 行业平均 PB | 衡量相对估值水平 |
| 相对 PS | 公司 PS ÷ 行业平均 PS | 衡量相对估值水平 |

### 3.3 盈利能力对比指标

| 指标 | 计算公式 | 用途 |
|------|---------|------|
| 毛利率排名 | 按毛利率从高到低排序 | 衡量产品竞争力 |
| 净利率排名 | 按净利率从高到低排序 | 衡量整体盈利能力 |
| ROE 排名 | 按 ROE 从高到低排序 | 衡量股东回报能力 |

### 3.4 成长性对比指标

| 指标 | 计算公式 | 用途 |
|------|---------|------|
| 营收增速排名 | 按营收增速从高到低排序 | 衡量收入增长能力 |
| 利润增速排名 | 按利润增速从高到低排序 | 衡量利润增长能力 |

### 3.5 综合竞争力评分

| 维度 | 权重 | 评分方法 |
|------|------|---------|
| 规模实力 | 20% | 市值、营收、利润排名百分比 |
| 估值吸引力 | 25% | PE、PB、PS 相对估值 |
| 盈利能力 | 25% | 毛利率、净利率、ROE 排名 |
| 成长性 | 20% | 营收增速、利润增速排名 |
| 财务健康 | 10% | 资产负债率、现金流排名 |

**综合得分计算**：
```
综合得分 = Σ(各维度得分 × 权重)
各维度得分 = 该维度各项排名百分比的平均值
```

## 四、分析逻辑与判断标准

### 4.1 行业地位判断

**按市值判断行业地位**：

| 市值排名 | 行业地位 | 说明 |
|---------|---------|------|
| 第 1 名 | 绝对龙头 | 行业最大公司 |
| 第 2-3 名 | 行业龙头 | 行业领先公司 |
| 第 4-10 名 | 挑战者 | 行业重要参与者 |
| 10 名以后 | 跟随者 | 行业中小公司 |

**按市占率判断**（如可获取）：

| 市占率 | 行业地位 | 说明 |
|--------|---------|------|
| >30% | 绝对龙头 | 行业主导地位 |
| 20-30% | 行业龙头 | 行业领先地位 |
| 10-20% | 重要参与者 | 行业主要竞争者 |
| 5-10% | 中小厂商 | 行业一般参与者 |
| <5% | 边缘厂商 | 行业边缘角色 |

### 4.2 估值对比分析

**相对 PE 判断**：

| 相对 PE | 评估 | 说明 |
|---------|------|------|
| <0.7 | 显著低估 | 估值显著低于行业 |
| 0.7-0.9 | 低估 | 估值低于行业 |
| 0.9-1.1 | 合理 | 估值与行业相当 |
| 1.1-1.5 | 高估 | 估值高于行业 |
| >1.5 | 显著高估 | 估值显著高于行业 |

**估值与成长匹配性**：
- 高估值 + 高成长：合理（高成长支撑高估值）
- 高估值 + 低成长：高估（估值泡沫）
- 低估值 + 高成长：低估（可能存在机会）
- 低估值 + 低成长：合理（价值股）

**估值与盈利质量匹配性**：
- 高 ROE 应享受高 PE 溢价
- 低 ROE 应给予低 PE 折价

### 4.3 盈利能力对比

**毛利率对比分析**：

| 毛利率排名 | 评估 | 说明 |
|-----------|------|------|
| 前 10% | 优秀 | 产品竞争力强 |
| 前 30% | 良好 | 产品竞争力较强 |
| 30-70% | 一般 | 产品竞争力中等 |
| 后 30% | 偏弱 | 产品竞争力较弱 |
| 后 10% | 差 | 产品竞争力弱 |

**ROE 对比分析**：

| ROE 排名 | 评估 | 说明 |
|---------|------|------|
| 前 10% | 优秀 | 股东回报能力极强 |
| 前 30% | 良好 | 股东回报能力较强 |
| 30-70% | 一般 | 股东回报能力中等 |
| 后 30% | 偏弱 | 股东回报能力较弱 |
| 后 10% | 差 | 股东回报能力弱 |

### 4.4 成长性对比

**营收增速对比**：

| 营收增速排名 | 评估 | 说明 |
|-------------|------|------|
| 前 10% | 高增长 | 增长远快于同行 |
| 前 30% | 较快增长 | 增长快于同行 |
| 30-70% | 中等增长 | 增长与同行相当 |
| 后 30% | 低速增长 | 增长慢于同行 |
| 后 10% | 负增长 | 收入下滑 |

**利润增速对比**：
- 判断标准同营收增速

**增速与规模匹配性**：
- 大公司通常增速较慢（基数大）
- 小公司通常增速较快（基数小）
- 大公司维持高增速更难能可贵

### 4.5 财务健康度对比

**资产负债率对比**：

| 资产负债率排名 | 评估 | 说明 |
|---------------|------|------|
| 最低 30% | 保守 | 负债率低，财务安全 |
| 30-70% | 合理 | 负债率适中 |
| 最高 30% | 激进 | 负债率高，风险较高 |

**注意**：不同行业负债率差异大，应与同行对比而非绝对值

## 七、注意事项

1. **同行选择**：确保选择真正可比的公司，细分行业、业务模式应相似
2. **数据口径**：确保各公司数据口径一致（如 TTM、年报等）
3. **特殊因素**：剔除有特殊因素的公司（如重组、亏损、异常值）
4. **行业周期**：周期股应对比周期位置，不同周期阶段估值不可比
5. **规模差异**：大公司与小公司估值逻辑不同，应分层对比
6. **业务多元化**：业务多元化的公司可能难以找到完全可比的同行
7. **上市地差异**：A 股、港股、美股估值体系不同，应分别对比
8. **动态变化**：行业格局可能快速变化，应定期更新对比数据
