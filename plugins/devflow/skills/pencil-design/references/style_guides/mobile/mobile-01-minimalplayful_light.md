# Minimal Playful Mobile — Style Guide

---

## Style Summary

### Description

A bright and approachable mobile interface that balances minimal foundations with playful pops of color. The design features a clean white canvas with coral accent (#FF6B6B) bringing warmth and energy to interactive elements. Bricolage Grotesque's distinctive character shapes add personality to headlines while DM Sans provides reliable readability for body content. Generous rounded corners (16-24px) create a friendly, approachable feel throughout. The palette stays restrained with white surfaces and subtle grays, letting the coral accent shine in buttons, indicators, and highlights. Perfect for lifestyle apps, wellness platforms, and consumer applications targeting a modern, optimistic audience.

### Key Aesthetics

- **Minimal Foundation:** Clean white canvas with subtle gray accents lets content breathe without visual noise.
- **Playful Coral:** Warm coral accent (#FF6B6B) brings energy and optimism to key interactive moments.
- **Distinctive Typography:** Bricolage Grotesque's unique letterforms add personality without sacrificing legibility.
- **Soft Geometry:** Generous rounded corners (16-24px) create an inviting, friendly tactile quality.
- **Restrained Palette:** White, grays, and coral—three elements create clear hierarchy without overwhelming.
- **Modern Simplicity:** Clean lines, ample whitespace, and purposeful color use for contemporary appeal.

## Color System

The palette embraces simplicity—white surfaces dominate, subtle grays provide structure, and coral (#FF6B6B) serves as the single warm accent for all interactive and status elements.

### Core Backgrounds

- #FFFFFF — Page Background, Card Surface (pure white canvas)
- #F8F9FA — Surface Subtle (light gray for subtle highlights)
- #F1F3F5 — Surface Muted (input backgrounds, inactive states)

### Text Colors

- #212529 — Text Primary (headlines, primary content)
- #495057 — Text Secondary (body text, descriptions)
- #868E96 — Text Muted (labels, placeholders, inactive)
- #ADB5BD — Text Disabled (disabled states)

### Accent Colors

- #FF6B6B — Coral Primary; CTAs, active states, links, highlights
- #FF8787 — Coral Light; hover states, lighter accents
- #FA5252 — Coral Dark; pressed states, emphasis
- #20C997 — Success; positive indicators, completion
- #51CF66 — Success Light; success backgrounds
- #CED4DA — Border; dividers, input strokes

## Typography

The dual-font system pairs Bricolage Grotesque's distinctive geometric character for headlines and UI with DM Sans's clean readability for body and data.

### Font Families

- **Bricolage Grotesque** — Headlines, titles, buttons, navigation; geometric grotesque with unique character shapes
- **DM Sans** — Body text, labels, data values, descriptions; clean, highly legible sans-serif

### Type Scale

- **36px** — Large Title. Bricolage Grotesque, bold 700, letterSpacing: -1
- **28px** — Section Title. Bricolage Grotesque, semibold 600
- **22px** — Card Title. Bricolage Grotesque, medium 500
- **18px** — Metric Value. DM Sans, bold 700
- **16px** — Body, Button. DM Sans, medium 500 / regular 400
- **14px** — Label, Caption. DM Sans, medium 500
- **12px** — Meta, Badge. DM Sans, semibold 600

### Font Weights

- **700** — Bold. Large titles, metric values
- **600** — Semibold. Section titles, badges
- **500** — Medium. Card titles, buttons, labels
- **400** — Normal. Body text, descriptions

### Letter Spacing

- **-1px** — Large titles (display compression)
- **0** — Default; all other text

### Line Height

- **1.5** — Body text; comfortable reading
- **1.2** — Tight; headlines, titles
- **1.0** — Single line elements

## Spacing System

### Gap Scale (between elements)

- **32px** — Major; section gaps
- **24px** — Large; card gaps, content sections
- **20px** — Medium; internal card gaps
- **16px** — Standard; row gaps, element groups
- **12px** — Small; icon to label, tight groups
- **8px** — Tight; button internal, small stacks
- **4px** — Minimal; meta stacks

### Padding Scale

- **[24, 20]** — Content wrapper, cards
- **[20, 16]** — Sections, lists
- **[16, 20]** — Buttons, inputs
- **[12, 16]** — Small cards, badges
- **[8, 12]** — Compact elements

### Layout Pattern

- **402px** — Screen Width (iPhone standard)
- **20px** — Content Padding (horizontal)
- **24px** — Section Gap (vertical)
- **16px** — Card Gap (within sections)

## Corner Radius

This design uses generous rounding throughout, creating a friendly, approachable aesthetic.

- **24px** — Large cards, modal sheets
- **20px** — Cards, containers
- **16px** — Buttons, inputs, small cards
- **12px** — Badges, chips
- **8px** — Small elements, icons
