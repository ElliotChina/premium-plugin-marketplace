# my-toolkit

Elliot 的个人工具箱插件，包含日常开发中常用的命令。

## 命令列表

| 命令 | 用途 |
|------|------|
| `/my-toolkit:project-config` | 项目配置 |
| `/my-toolkit:new-feature` | 新功能开发 |
| `/my-toolkit:fix` | 问题诊断修复 & 功能调试 |
| `/my-toolkit:test` | 功能测试 |
| `/my-toolkit:commit` | 代码提交 |
| `/my-toolkit:docs` | 更新文档 |
| `/my-toolkit:prd` | 写需求文档 |
| `/my-toolkit:review` | 代码&文档审查 |
| `/my-toolkit:ui` | 原型/UI 设计 |
| `/my-toolkit:pencil-design` | Pencil MCP UI 设计 |

## 安装

```bash
claude plugin add /path/to/my-toolkit
```

## 使用

在 Claude Code 中直接输入命令即可，例如：

```
/my-toolkit:new-feature 用户登录功能
/my-toolkit:fix 页面加载时报 500 错误
```
